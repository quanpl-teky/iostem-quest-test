Pond Documentation

Angles

The compass is oriented so that due east (right) is 0 degrees, 90 is north, 180 is west, and 270 is south. Angles wrap as expected: -90 is south, 450 is north.

API Functions

cannon(angle, range)

Fires a cannonball towards the specified angle and range. The angle is independent of the duck's direction. The range can be from 0-70 meters.

// Fire east, at a range of 50 meters.
cannon(0, 50);

scan(angle, resolution)

Activates the duck's radar. Returns the range to the nearest opponent in a given direction. If no opponent is found, it returns Infinity. The optional resolution parameter (1-20 degrees, default 5) specifies the width of the scanning beam.

var distance = scan(90); // Scan north
if (distance < 70) {
  cannon(90, distance);
}

swim(angle, speed)

Starts the duck moving in the specified direction. It will continue moving indefinitely. The optional speed parameter can be from 0 (stopped) to 100 (fast), with a default of 50.

// Swim towards the north-west at full speed.
swim(135, 100);

stop()

Stops the duck from moving. The duck will slow to a stop.

stop();

getX() and getY()

Return the duck's horizontal (X) and vertical (Y) positions. Values are between 0 and 100, starting from the lower-left corner.

speed()

Returns the duck's current speed (0-100).

health()

Returns the duck's current health (0 is sunk, 100 is perfect).

damage()

Returns the duck's current damage (0 is perfect, 100 is sunk). This is equivalent to 100 - health().

Control Structures

if / else if / else

Executes code conditionally.

if (getX() < 10) {
  // Too close to the left wall.
  swim(0, 50); // Go east.
} else {
  // Safe.
  stop();
}

while (condition)

Repeats a block of code as long as the condition is true.

// Scan in a circle until an enemy is found.
var angle = 0;
while (scan(angle) == Infinity) {
  angle = angle + 10;
}
cannon(angle, scan(angle));

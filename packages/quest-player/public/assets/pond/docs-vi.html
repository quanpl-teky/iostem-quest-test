Tài liệu Game Ao hồ

Góc độ

La bàn được định hướng sao cho hướng đông (phải) là 0 độ, 90 là hướng bắc, 180 là hướng tây, và 270 là hướng nam. Các góc sẽ tự động quay vòng như mong đợi: -90 độ là hướng nam, 450 độ là hướng bắc.

Các hàm API

cannon(goc, pham_vi)

Bắn một viên đạn pháo theo góc và phạm vi đã chỉ định. Góc bắn độc lập với hướng của con vịt. Phạm vi có thể từ 0-70 mét.

// Bắn về hướng đông, ở phạm vi 50 mét.
cannon(0, 50);

scan(goc, do_phan_giai)

Kích hoạt radar của con vịt. Trả về khoảng cách tới đối thủ gần nhất theo một hướng nhất định. Nếu không tìm thấy đối thủ, nó sẽ trả về Infinity. Tham số do_phan_giai (tùy chọn, từ 1-20 độ, mặc định là 5) chỉ định độ rộng của tia quét.

var khoang_cach = scan(90); // Quét về hướng bắc
if (khoang_cach < 70) {
  cannon(90, khoang_cach);
}

swim(goc, toc_do)

Bắt đầu cho vịt di chuyển theo hướng đã chỉ định. Nó sẽ tiếp tục di chuyển vô thời hạn. Tham số toc_do (tùy chọn) có thể từ 0 (dừng) đến 100 (nhanh), với giá trị mặc định là 50.

// Bơi về hướng tây bắc với tốc độ tối đa.
swim(135, 100);

stop()

Dừng việc di chuyển của con vịt. Con vịt sẽ từ từ dừng lại.

stop();

getX() và getY()

Trả về vị trí theo chiều ngang (X) và chiều dọc (Y) của con vịt. Các giá trị nằm trong khoảng từ 0 đến 100, bắt đầu từ góc dưới bên trái.

speed()

Trả về tốc độ hiện tại của con vịt (0-100).

health()

Trả về mức máu hiện tại của con vịt (0 là bị chìm, 100 là hoàn hảo).

damage()

Trả về mức sát thương hiện tại của con vịt (0 là hoàn hảo, 100 là bị chìm). Tương đương với 100 - health().

Cấu trúc điều khiển

if / else if / else

Thực thi mã theo điều kiện.

if (getX() < 10) {
  // Quá gần bức tường bên trái.
  swim(0, 50); // Đi về hướng đông.
} else {
  // An toàn.
  stop();
}

while (dieu_kien)

Lặp lại một đoạn mã miễn là điều kiện còn đúng.

// Quét theo vòng tròn cho đến khi tìm thấy kẻ thù.
var goc = 0;
while (scan(goc) == Infinity) {
  goc = goc + 10;
}
cannon(goc, scan(goc));
